## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/ASP.NET-Landing-Page.git
```

### 2️⃣ Navigate to the project folder

```
cd ASP.NET-Landing-Page
```

### 3️⃣ Restore dependencies

```
dotnet restore
```

### 4️⃣ Run the application

```
dotnet run
```

---

## 🌐 Running the Application

After running the project, open your browser and go to:

```
http://localhost:5032
```

The landing page will be displayed.

## 📦 Deployment

You can deploy this ASP.NET Core project using:

* **IIS (Internet Information Services)**
* **Azure App Services**
* **Docker Containers**
* **Render / Cloud Hosting**

Publish the project using:

```
dotnet publish -c Release -o ./publish
```
